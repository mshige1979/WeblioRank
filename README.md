WeblioRank
==========

単語の順位を表示するだけのWebアプリ

### 環境
- Linux(CentOS)
- Perl
- MySQL

### ツール
- compass
- PhpStorm
- git
- Sequel Pro

### フレームワーク
- Pure
- knockout.js
- Mojolicious

### 作成環境
- 開発（vagrant）
- 本番（vps）

### アプリケーションサーバ環境
- supervisor
- apache
- Starman

### 環境変数
```
export WEBSEARCHAPP_DATABASE="db_name"
export WEBSEARCHAPP_HOST="server"
export WEBSEARCHAPP_USER="foofoo"
export WEBSEARCHAPP_PASS="hogeohge"

```
